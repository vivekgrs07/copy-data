## Dependencies Types/6 scopes of dependencies

   1. compile (Default scope)   --> Ex spring core,log4j ---> Available on compile time and run time.
   2. runtime ()                --> e.g mysql-connector, postresql-driver    (Not required on compile time, required on runtime)
   3. test                      --> e.g Junit, Selenuim, Test NG  --> Only for testing purpose.
   4. provided   	        --> e.g Servelet API, JDK Libraries  --> Available on compile time but it provides environment on runtime.
   5. system                    --> e.g provided from local repository not from maven central repo.
   6. import {very importent}	-->  Import other project dependency (pom.xml) in your project (pom.xml).
